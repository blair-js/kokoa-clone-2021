*반복문* loop iterate 인간은 반복적인 작업을 잘하지 못한다. 컴퓨터는 이런 반복적인 작업을 대행하기 위해서 만들어진 기계다. 반복문은 컴퓨터에게 반복적인 작업을 지시하는 방법이다. *반복문의 문법* 1) while while (조건.불린) { 반복해서 실행할 코드 } 초기화; while(반복조건) { 반복해서 실행할 코드 반복이 될 때마다 실행되는코드; } 2) for for (초기화; 반복조건; 반복이 될 때마다 실행되는 코드) { 반복해서 실행할 코드 } *tip* i = i + 1 과 i++ 는 같은 의미 * ++i(전위전산자) 와 i++(후위전산자) 의 차이점 * 둘다 i 변수에 1을 더하는 방식이지만 약간의 차이가 있다. ++i : 먼저 증가하고 작업이 수행 (바로 i의 값에 1이 더해진 값이 실행되며 시작) i++ : 작업이 수행된 후 값이 증가 (i의 값이 먼저 실행되고 두번째부터 1이 더해진 값으로 실행) *반복문의 제어* break : 특별한 조건을 만나면 반복을 끝내라. 이제 더 이상 반복X. 반복작업을 중간에 중단시키고 싶다면 break 사용! continue : 반복을 끝내지 않는다. 반복문(while, for문)의 {} 안에서 continue 문장을 만난 순간 continue 아래에 있는 실행해야 하는 문장들을 건너 뛰고, 다음 반복을 시작한다. ===> 즉 break는 제어흐름을 중단하고 빠져나오지만, continues 제어흐름을 유지한 상태에서 코드의 실행만 건너뛰는 역할을 한다. (반복문 다음반복을 시작하러 감) *반복문의 중첩* 반복문 안에 반복문이 들어가 있는 형태. 밖에 있는 반복문이 1번 실행될 때마다 안에 있는 반복문이 모든 반복을 실행하게 된다. 그리고 안에있는 반복문의 동작이 완료되면 다시 밖에 있는 반복문이 새로운 반복을 시작한다. 예를들면 밖에 있는 반복문이 총 5번을 반복하고, 안에 있는 반복문이 3번을 반복하는 형태라면 총 15번의 동작이 실행된다는 것!
