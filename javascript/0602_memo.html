1) this 함수 내에서 함수 호출 맥락을 의미한다. 맥락이라는 것은 상황에 따라서 달라진다는 의미인데 즉 함수를 어떻게 호출하느냐에 따라서 this가 가리키는 대상이 달라진다는 뜻이다. ex. function func() { if(window === this) { document.write("window === this"); } } func(); => 결과 window === this var o = { func : function() { if( o === this) { document.write("o === this"); } } } o.func(); => 결과 o === this
