JavaScript 란? 웹페이지에 생동감을 불어넣기 위해 만들어진 프로그래밍 언어이다. 동적인 언어. (HTMl,CSS는 정적인 언어:브라우저를 통해 웹페이지를 화면에 그려주면 이 화면을 변경할 방법이 없다는 의미) 웹브라우저를 제한하기 위한 언어. 웹브라우저: 서버와 클라이언트 사이의 인터페이스 역할을하며 웹 문서를 클라이언트에 표시하는 소프트웨어. (ex.익스플로러,safari,chrome 등) 웹서버: 웹 응용 프로그램을 유지 관리하고 응답을 생성하며 클라이언트 데이터를 승인하는 소프트웨어 및 시스템. -> 사용되는 대표적인 기술 : PHP,JAVA,PYTHON... Node.js JavaScript를 브라우저에서만 쓰는 것이 아닌 브라우저 밖. 즉, 내 컴퓨터에서 다양한 용도로 확장하기 위해 만들어진 기술. 탈웹브라우저!!! 각광받는 기술. 웹브라우저를 제어하기위한 JavaScript 기술 웹서버를 제어하기위한 JavaScript 기술
